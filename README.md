# eazy_net
## 项目介绍
    eazy_net是一个用于学习Java web方向项目，采用传统SSM + JSP开发，后期将拓展为最新技术。
    作者：周先生、孙先生
    
### 项目技术栈

#### 后端技术栈
    1、Spring
    2、Spring MVC
    3、Spring Security(待开发)
    4、Mybatis
    5、...
    
#### 前端技术栈
    1、JSP
    2、Bootstrap
    3、Jquery
    4、...
